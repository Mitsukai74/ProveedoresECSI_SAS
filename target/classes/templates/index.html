Gestión de provedores para ECSI SAS

Instructivo:

Siga los siguietes pasos para sacarle provecho a la aplicación:


En esta pantalla se puede apreciar la tabla con el listado de los proveedores creados hasta el momento con información como por ejemplo con el número de información tributaria, el número asignado a cada proveedor cuando este fue creado, el número de actividad económica y otros campos releventes.
Con esta funcionalidad, se le brinda al usuario la posibilidad que con unas pocas palabras o con el nombre exacto, pueda realizar un filtro dentro de la tabla y aparezcan solamente el que desea. Se puede buscar por una o varias letras clave que deben estar, dentro de la razón social del proveedor a buscar.
En cuanto a la creación de proveedores, se creó un formulario para que el usuario pueda generar un proveedor completamente nuevo con todos sus respectivos campos los cuales, están validados para que dentro de la base de datos, se pueda crear sin ningún inconveniente. En cuanto la edición, se le brinda al usuario la posibilidad de editar a un proveedor creado previamente.
